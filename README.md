# a16z New Media 2026 Summer Reading Reports

This project turns a saved local copy of **a16z New Media's 2026 Summer Reading List** into a reader-facing HTML guide for deciding which books to read when time is limited.

## Live page

Read the rendered report here:

https://az9713.github.io/summer-reading-list-2/

## What is included

- `index.html` - the GitHub Pages entrypoint for the main tabbed HTML report.
- `summer_reading_list_reports.html` - the same report kept under its descriptive filename.
- `01_moby_dick.md` through `10_bedes_ecclesiastical_history.md` - background markdown research notes for each book.
- A **Reading Order** tab tailored to Silicon Valley / San Francisco AI readers, ranking the books by likely usefulness and attention cost.

## Source

The starting source was the local MHTML capture:

`a16z New Media's 2026 Summer Reading List.mhtml`

That capture was used to identify the book list and the a16z recommendation rationale for each title. The individual reports add external research sources listed inside each book report. The raw MHTML capture is not committed because it is a local saved article capture; the repository contains the derived research and presentation artifacts.
